# File Organizer (Python)

A simple Python script that automatically organizes files into folders based on file type.

## Features
- Sorts images, documents, videos, etc.
- Automatically creates folders
- Easy to customize

## How to Use
1. Place files inside:
   file_organizer/files/
2. Run:
   python organizer.py

## Example
Before:
files/
  image.png
  notes.txt

After:
files/
  Images/image.png
  Documents/notes.txt
